  • MBInAppMessageManager

    This class manages the presenting and the queue of in app messages. It’s called by the plugin when the messages are checked after the delay setted. This class saves when a message is showed and display only the messages that are not showed (if debug is disabled) It manages also the queue of messages and finds out the main UIViewController where messages will be showed
