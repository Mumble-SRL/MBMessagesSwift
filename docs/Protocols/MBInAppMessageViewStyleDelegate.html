MBInAppMessageViewStyleDelegate

public protocol MBInAppMessageViewStyleDelegate : AnyObject

This protocol can be used to use different colors and fonts to the in-app messages views. Returning values from those functions will override the defults and the colors and fonts setted in the dashboard.

  • backgroundStyle(forMessage:) Default implementation

    The background style that will be used for the message. The default value is MBInAppMessageViewBackgroundStyle.translucent for banner messages, MBInAppMessageViewBackgroundStyle.solid otherwise
